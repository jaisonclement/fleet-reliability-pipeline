# fleet-reliability-pipeline
End-to-end Python pipeline for aviation maintenance data cleaning, validation, recurring defect analysis, and PDF/Excel reliability reporting.
# Aviation Reliability Portfolio Projects

This repository contains two connected aviation reliability projects:

## 1. Python Reliability Control Pipeline

Folder: `01_python_reliability_pipeline`

This project simulates aviation maintenance defect data, cleans it with Python and pandas, validates missing/duplicate records, analyzes recurring defects, and exports PDF/Excel reliability-control reports.

Key outputs:

- Cleaned aviation defect CSV
- Validation flag report
- Recurring defect frequency analysis
- System and aircraft downtime summaries
- PDF reliability report
- Excel dashboard workbook
- VBA macro module for report export/highlighting
